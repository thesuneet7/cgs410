# Empirical Distribution of Intervener Complexity

Course project for **CGS410 — Language in the Mind and Machine**.

For each dependency arc `head → dependent` in 12 Universal Dependencies
treebanks, we extract the **intervening tokens** (any token whose linear
position lies strictly between the head and the dependent) and quantify
their complexity along four features:

| Feature | What it measures | Where it is computed |
|---------|------------------|----------------------|
| A | UPOS (part-of-speech) of the intervener | `intervener_upos` |
| B | **Arity** — number of direct UD children of the intervener | `intervener_arity` |
| C | **Subtree size** — number of tokens in the subtree rooted at the intervener | `intervener_subtree_size` |
| D | **Attachment type** — whether the intervener attaches to the arc head, the arc dependent, or to neither | `attachment_type` |

A null distribution is generated by **randomly permuting word order** while
keeping the dependency tree fixed. The real distribution is then compared
to this baseline using **Jensen–Shannon divergence** and a one-tailed
**Mann–Whitney U test** (Bonferroni-corrected for 12 languages).

Reference paper: Yadav, Mittal & Husain (2022). *A Reappraisal of Dependency
Length Minimization as a Linguistic Universal*, **Open Mind 6**, 147–168.
[doi:10.1162/opmi_a_00060](https://doi.org/10.1162/opmi_a_00060).

## Languages (12)

English, German, Spanish, Russian, Arabic, Hindi, Turkish, Finnish,
Japanese, Chinese, Basque, Ancient Greek.

## Setup

```bash
pip install -r requirements.txt
```

The CoNLL-U treebanks must already be present under `./data/raw/<UD_*>`.
This repo ships them so no download is needed.

## Run order

```bash
# (1) verify the treebanks are valid CoNLL-U
python3 step2_verify_data.py

# (2) extract every (arc, intervener) observation across all 12 languages
python3 step3_extract_interveners.py

# (3) aggregate features (POS / arity / subtree / attachment + summary stats)
python3 step4_compute_features.py

# (4) random-linearisation null baseline (capped at 20k sents/lang, 1 perm)
python3 step5_baseline_generator.py

# (5) JSD + Mann-Whitney U + head-rate comparison
python3 step6_statistical_tests.py

# (6) all 10 figures for the report
python3 step7_visualizations.py
```

Each step prints exactly which command to run next.

## Output layout

```
data/
  raw/                            # input UD treebanks (already present)
  processed/                      # step 3 — per-language interveners CSVs
  aggregated/                     # step 4 — distribution tables + JSON
  baseline/                       # step 5 — random-linearisation null
  results/                        # step 6 — statistical tests + report
  figures/                        # step 7 — 10 PDF + PNG figures
```

```bash
python3 test_extraction.py     # unit tests for the extractor
```
